 Healthcare Data Analytics Dashboard (Excel)

 Overview

The **Healthcare Data Analytics Dashboard** is an Excel-based project that analyzes patient data to uncover insights related to demographics, admissions, and healthcare trends. The dashboard provides a clear and interactive view of key metrics to support data-driven decision-making in healthcare management.

---

 Objectives

* To analyze patient and hospital data using Excel
* To identify trends in admissions, age groups, and services
* To create an interactive and user-friendly dashboard
* To derive insights for improving healthcare operations

---

 Tools & Technologies

* **Microsoft Excel**

  * PivotTables
  * Advanced Formulas
  * Charts & Visualizations
  * Data Cleaning

---

 Key Features

* Interactive dashboard for easy data exploration
* Analysis of patient demographics (age, gender, etc.)
* Admission trends and patterns
* Department/service-wise distribution
* Dynamic charts and summary metrics

---

 Insights Generated

* Identified patterns in patient admissions over time
* Analyzed most utilized healthcare services
* Highlighted key demographic trends
* Supported decision-making for resource allocation

---

 Outcomes

* Strengthened skills in Excel data analysis and visualization
* Improved ability to transform raw data into actionable insights
* Gained hands-on experience in dashboard creation

---

 Conclusion

This project demonstrates how Excel can be effectively used to analyze healthcare data and build dashboards that provide meaningful insights for better planning and management.
